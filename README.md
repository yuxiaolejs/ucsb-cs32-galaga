## General Documentation
### System Requirements
- Platform: Any that supports OpenGL, FreeGLUT, gcc, cmake
- OS: Linux recommanded, but any system should be able to run.
- CPU: 2 Core or more
- Memory: 4 GB or more
- Disk: 100 MiB or more
- Software


  | name     | version |
  | -------- | ------- |
  | CMake    | ^3.14   |
  | OpenGL   | 4.1     |
  | FreeGLUT | ^3.4.0  |
  | libpng   | -       |


### Compiling the program
```
cd ucsb-cs32-galaga
cmake .
make
```
### Running the program
```
./ezg
```
